E-commerce Orders Data Visualization Project

"Exploring Sales Trends and Insights through Data Visualization"

The E-commerce Orders dataset consists of customer order details including product categories, regions, and sales data. The goal of this project was to visualize key trends and insights, using Pythonâ€™s visualization libraries such as Seaborn and Matplotlib.

The dataset includes the following features:

  • Order ID: Unique identifier for each order
  • Order Date: The date the order was placed
  • Ship Date: The date the order was shipped
  • Customer ID: Unique identifier for the customer
  • Segment: Type of customer (Consumer, Corporate, Home Office)
  • City: City where the order was placed
  • State: State where the order was placed
  • Region: Region (East, West, Central, South)
  • Product Category: Office Supplies, Furniture, Technology
  • Sub-Category: Specific sub-group of products
  • Sales: Total dollar amount of the order
  • Quantity: Number of items ordered
  • Discount: Discount applied to the order
  • Profit: Profit made on the order

Data Cleaning and Preprocessing: We began by converting date columns to datetime objects, checking for any missing or duplicate values, and adding new features like "Order Month" and "Order Year". This helped set up the data for analysis and visualization.

Key Visualizations: Throughout this project, various plots and graphs were created to gain insights:

  • Monthly Sales Trend: Shows sales trends across different months of the year.
  • Category and Sub-category Analysis: Displays total sales and profit by category and sub-category.
  • Regional Sales Comparison: A pie chart showing the sales distribution by region.
  • Ship Mode Usage: Shows how often each shipping mode is used.
  • Discount vs Profit: A scatter plot exploring the relationship between discounts and profit.
  • Heatmap of Correlations: A heatmap showing correlations between key numeric columns.
  • Top Cities by Sales: Displays a bar chart of top cities with the highest total sales.

Conclusion: The E-commerce Orders Data Visualization project helped uncover several valuable insights:

  • The Technology category generated the highest revenue.
  • Some sub-categories like Tables had negative profits despite high sales.
  • The West and East regions contributed most to overall sales.
  • Heavy discounts often resulted in reduced profits.
  • Standard Class was the most frequently used shipping mode.

Tools Used: The analysis was conducted using Python in Google Colab, with libraries like Pandas for data manipulation, Seaborn and Matplotlib for visualization, and NumPy for numerical operations.

Next Steps: Future steps include performing customer segmentation based on order patterns, predicting future sales using machine learning, and deploying interactive dashboards.

Value of This Project: This project showcases my skills in data cleaning, exploration, and visualization. It also highlights the ability to extract business insights from complex datasets, which is a key skill for any data analyst.
