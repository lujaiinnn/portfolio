Insurance Dataset Regression Models

"Exploring Medical Insurance Costs with Regression Models"

The Insurance dataset consists of various features like age, BMI, number of children, smoking status, and region that can be used to predict the insurance charges for an individual. In this blog, we apply multiple regression models to predict medical insurance charges using the dataset's features.

The regression models we use include:

  • Linear Regression
  • Polynomial Regression
  • Decision Tree Regression
  • Random Forest Regression

We start by preprocessing the data, where we handle categorical variables like sex, smoker status, and region by converting them into numeric form using one-hot encoding. This transformation makes the data suitable for machine learning models.

Linear Regression: A baseline regression model is applied to predict insurance charges based on the input features. The results are used as a benchmark for comparing more complex models.

Polynomial Regression: A non-linear approach is applied to capture the relationship between the features and the target variable (charges) more accurately.

Decision Tree Regression: A decision tree model is used to create a tree structure for predicting insurance charges, and we also tune hyperparameters for better accuracy.

Random Forest Regression: A more powerful model that aggregates the predictions of multiple decision trees, helping to reduce overfitting and improve accuracy. Hyperparameter tuning is also applied for optimal results.

Evaluation: We evaluate the performance of each model using metrics such as Mean Absolute Error (MAE), Mean Squared Error (MSE), and the RÂ² score. Based on these metrics, we conclude that the Random Forest Regression model performs the best, with an RÂ² score of 86.17%, followed by Polynomial Regression.

Conclusion: The Random Forest model with hyperparameter tuning is the most effective at predicting insurance charges. However, Polynomial Regression also provides useful insights into the non-linear relationships between the features and target variable.
