Image Classification Using SVM: Horse vs Human

"Differentiating Horses and Humans with Support Vector Machines"

In this project, we applied traditional machine learning methods to classify images of horses and humans. We used a Support Vector Machine (SVM) model, which is effective for binary classification tasks, and implemented it using Python. Our dataset consisted of images labeled as either 'horse' or 'human'. We preprocessed the images, trained an SVM model, and evaluated its performance. The goal was to achieve a high accuracy in distinguishing between the two categories.

The steps followed in the project include:

  • Image Preprocessing
  • Model Training using Support Vector Machine (SVM)
  • Model Evaluation
  • Prediction on new images

Image Preprocessing: The images were resized to 64x64 pixels and flattened to one-dimensional arrays for input into the SVM model. Labels were assigned: '0' for horses and '1' for humans. The dataset was split into training and testing sets in an 80-20 ratio.

Support Vector Machine (SVM): We used the Support Vector Classification (SVC) model from scikit-learn with a linear kernel. The model attempts to find the best hyperplane that separates the two classes with the maximum margin.

Model Evaluation: We evaluated the model using accuracy, confusion matrix, and classification report. The model achieved an accuracy of around 91%, with balanced precision and recall for both classes.

Prediction on New Images: We tested the trained model on new images not included in the training or test sets. The images were resized and preprocessed in the same way before being passed to the model for classification. The predicted label was displayed along with the image.

Challenges Encountered: During implementation, we faced some issues with image dimension mismatches and class labeling errors. These were resolved by ensuring consistent preprocessing and checking the order of the classes when labeling images.

Future Improvements: Future work could include using Convolutional Neural Networks (CNNs) for better feature extraction, data augmentation to increase the dataset size, and creating a web application to allow users to upload and classify images on the fly.

Conclusion: This project demonstrates the effectiveness of SVM for simple image classification tasks. Although not as powerful as CNNs, SVM offers a simple and interpretable solution that can be applied to binary image classification problems.
