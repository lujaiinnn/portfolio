Sales Data Visualization Project

"Turning Sales Numbers into Strategic Business Insights"

This project aims to analyze a sales dataset using Python's data visualization libraries. The dataset simulates retail sales and includes fields like date, region, product, units sold, unit price, and total sales. We used pandas, matplotlib, and seaborn to clean, transform, and visualize the data for meaningful business analysis.

Data Fields:

  • Date: Transaction date
  • Region: North, South, East, West
  • Product: Product name
  • Units Sold: Quantity sold
  • Unit Price: Price per item
  • Total Sales: Calculated by Units Sold Ã— Unit Price

Data Preparation: We cleaned the dataset by checking for missing values, converting the 'Date' column to datetime format, and creating a derived column 'Total Sales'.

Visualizations: We applied several visual techniques to uncover sales patterns:

  • Line Plot: Sales trends over time for each product using seaborn's lineplot.
  • Stacked Bar Chart: Product-wise sales by region using pandas groupby and unstack.
  • Scatter Plot: Unit Price vs Total Sales, highlighting correlations and outliers.
  • Count Plot: Frequency of transactions per region using seaborn.
  • KDE Plot: Sales distribution for each product, visualizing density and spread.
  • Pie Chart: Proportional contribution of each region to total sales.
  • Bar Chart: Average total sales by day of the week to identify peak selling days.

Insights Gained: The Western region dominated sales volume, with Friday and Saturday showing the highest revenue. Mid-priced products delivered higher overall revenue. A steady sales increase was observed throughout the year.

Conclusion: Data visualization helped uncover key trends in sales performance across products, time periods, and geographic segments. The project showcases practical use of data wrangling and exploratory analysis skills, making it a valuable addition to any data analyst's portfolio.

Tools Used: Python (Jupyter/Colab), pandas, matplotlib, seaborn, NumPy.

Future Scope: We plan to expand the project by adding predictive forecasting, building dashboards using tools like Plotly or Tableau, and automating reports using APIs or scheduled scripts.
