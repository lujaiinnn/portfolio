Glass Identification Project

"Unveiling Hidden Structures in Chemical Data with Unsupervised Learning"

The Glass Identification dataset is a popular benchmark dataset used in pattern recognition and classification. The goal is to identify the type of glass (e.g., window float, container, etc.) based on its chemical properties such as sodium, magnesium, aluminum, and others. This classification task can benefit from unsupervised learning techniques to uncover hidden patterns and natural groupings without using the label (Type) during training.

In this blog, we explore a range of unsupervised learning techniques including K-Means Clustering, Hierarchical Agglomerative Clustering, DBSCAN, PCA (Principal Component Analysis), and LDA (Linear Discriminant Analysis, used here for visualization).

Before applying any algorithm, the data must be preprocessed. Since unsupervised learning relies heavily on distance metrics, standardization is crucial. Using StandardScaler, we normalize all features (except the 'Type' label) to ensure fair clustering.

K-Means Clustering: This model separates data into 'k' clusters by minimizing the Within-Cluster Sum of Squares (WCSS). We used the Elbow Method to determine that 6 clusters work best. The results are visualized using PCA for intuitive grouping.

Hierarchical Clustering: Agglomerative clustering begins with individual points and merges them based on proximity. A dendrogram helps determine 3 optimal clusters. Again, PCA projections allow easy interpretation.

DBSCAN: Unlike previous models, DBSCAN finds clusters based on density. It labels outliers as noise and is great for irregular shapes. The parameters epsilon and min_samples are tuned, revealing distinct groups and noise points in the dataset.

PCA: Principal Component Analysis reduces dimensionality for visualization. It preserves variance and simplifies complexity. Our PCA plots show that most data variance lies within the first two principal components.

LDA: Though a supervised method, we use LDA here to visualize how well unsupervised clusters match actual class labels. It provides insight into class separability and validates cluster quality.

Conclusion: Each unsupervised method offered a unique perspective. K-Means formed clean spherical groups, Hierarchical clustering allowed dendrogram insights, and DBSCAN managed arbitrary shapes and noise. PCA and LDA provided powerful visualization tools. Together, these techniques form a complete unsupervised learning pipeline for glass classification based solely on chemical composition.
